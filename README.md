# Job Salary Prediction Model
# Overview
This project predicts employee salaries using Machine Learning based on factors such as:
- Job Title
- Education Level
- Industry
- Company Size
- Location
- Work Mode (Remote/Hybrid/On-site)
- Years of Experience
- Skills Count
- Certifications

# Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit

# Features
- User-friendly Streamlit web application
- Real-time salary prediction
- Machine Learning based model
- Interactive input form
